# Practical Explainer Plate Seed

User example: useful searchable images like a "Types of Coffee" visual guide or "The Nitrogen Cycle" educational diagram.

## Desired Direction

Create practical, high-SEO visual explainers that answer real search queries directly. These should be polished reference posters, comparison charts, labeled diagrams, and course/training handouts, not random decorative images.

## Niche Audiences

- Teachers and curriculum authors.
- Students searching for visual study guides.
- Trainers making internal onboarding materials.
- Creators and designers needing reference plates.
- Agents retrieving topic-specific visual knowledge.

## Search Intent Examples

- `types of coffee chart`
- `coffee drinks visual guide`
- `nitrogen cycle diagram`
- `biology cycle infographic`
- `visual guide to common coffee drinks`
- `science process labeled diagram`
- `classroom revision poster`
- `training poster for cafe staff`

## Useful Formats

- Labeled taxonomy chart.
- Process cycle diagram.
- Comparison grid.
- Anatomy or parts plate.
- Step-by-step method poster.
- Course revision sheet.
- Training reference poster.

## Quality Notes

- The image should have a clear title and immediately readable structure.
- Use concise labels and short captions.
- Prefer real-world examples and visual anchors.
- Store the exact intended labels/captions in the prompt card.
- Treat exact generated text as risky; verify or mark exploratory when needed.
- Optimize metadata for both Google indexing and local agentic image search.
