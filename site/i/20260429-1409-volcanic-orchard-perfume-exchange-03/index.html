Learning Value

Push the corpus into a fashion lookbook format while keeping the wardrobe embedded in an actual working economy.

Composition Notes

The lineup needs clean silhouette separation and a shared garment language so it reads as one occupational world rather than six random outfits.

Prompt Extract

Use case: stylized-concept
Asset type: fashion lookbook plate for a speculative fragrance economy
Primary request: a six-look lineup of harvest porters, scent brokers, blotter samplers, bottle couriers, roof growers, and exchange clerks from the volcanic orchard perfume world
Scene/backdrop: plain mineral studio backdrop with a faint gradient and floor shadow, no props except a few restrained trade accessories
Subject: full-body fashion looks with layered weatherproof capes, aprons, bottle harnesses, ash veils, ledger belts, cooled carrying frames, sulfur-cream gloves, and fruit-stain details; each look clearly distinct but from one system
Style/medium: editorial fashion lookbook photography, sharp and premium but grounded, not costume design sheets
Composition/framing: straight-on lineup of six standing figures across a wide frame, generous spacing, each silhouette fully visible from head to boot
Lighting/mood: soft directional studio light with subtle cool haze and disciplined contrast
Color palette: charcoal, olive, ash cream, burgundy wax, ember orange, weathered copper, smoky plum
Materials/textures: waxed canvas, wool, netted veils, brushed brass, rubberized hems, frosted bottle glass, raw linen, singed leather
Constraints: original wardrobe system only; no logos; no watermark; no labels; avoid runway glamour excess; make the garments utilitarian, ceremonial, and class-co
