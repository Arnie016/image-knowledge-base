Prompt Extract

Use case: infographic-diagram
Asset type: event-floor plan and seating diagram for a speculative public roller disco
Primary request: an overhead architectural event plan of the sea-glass bathhouse roller disco showing the terrazzo oval rink, balcony standing zones, shallow reflecting channels, rental desk, citrus soda bar, organ booth, repair alcove, beginner lane, and checkerboard locker galleries
Scene/backdrop: clean cream drafting sheet with subtle paper grain and restrained color blocks
Subject: a top-down bathhouse plan with crisp spatial hierarchy, simple legend chips, arrows, zone labels, and clear circulation paths
Style/medium: polished diagrammatic print layout, halfway between architectural plan and public event program
Composition/framing: full-page overhead plan centered with supporting legend blocks around the edges
Lighting/mood: flat even lighting as if photographed from above on a desk
Color palette: cream paper, aqua fills, coral route lines, navy labels, citron accents, terrazzo fleck neutrals
Materials/textures: ink on coated paper, pencil traces, light halftone blocks, taped corner marks
Text (verbatim): "RINK", "BALCONY", "RENTALS", "ORGAN", "SODA", "LOCKERS", "BEGINNER LANE", "REPAIR"
Constraints: clean readable hierarchy; no logos; no watermark; avoid overloading with tiny text; make the plan feel civic and used, not corporate branding
Avoid: sci-fi HU
