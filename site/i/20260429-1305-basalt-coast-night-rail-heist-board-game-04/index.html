Use case: ui-mockup
Asset type: digital companion app screen for a tabletop strategy game
Primary request: a shippable mobile and tablet companion interface for the fictional board game Basalt Coast Night-Rail Heist
Scene/background: screen-only product mockup with no hands, focused on the UI itself
Subject: central tactical map of a fogbound cliff rail network, one compact objective strip, one danger meter, route cards, cargo and alarm tracks, and a collapsible dossier drawer
Style/medium: polished browser-game companion UI, thematic but functional, not a generic SaaS dashboard
Composition/framing: portrait tablet screen centered, central map protected, persistent HUD limited to the edges, one small secondary drawer partially open
Lighting/mood: low-light premium screen glow with crisp contrast and restrained depth
Color palette: coal black, smoky blue, brass lines, cream typography, dark burgundy alerts
Materials/textures: frosted glass panels, etched brass dividers, subtle paper-card textures, matte dark UI chrome
Constraints: make it feel playable and readable; minimal concise text only; no oversized boxed panels; keep the center mostly clear; original interface design only
Avoid: generic productivity dashboard; game HUD clutter; neon cyberpunk; huge headers; full-width top-and-bottom chrome; watermark
