Test whether a human lineup can deepen the world without drifting into fashion-editorial habits that have already appeared in other runs.

Composition Notes

The figures needed uniform spacing and workmanlike posture so the badge, cape, satchel, and orange release trim read as system cues rather than style flourishes.

Prompt Extract

Use case: mixed-format
Asset type: caretaker uniform lineup
Primary request: six funeral salon attendants for retired weather balloons presented as a disciplined service lineup rather than fashion editorial
Scene/background: simple painted wall with floor marks and a hanging cloud-emblem banner, studio-documentary setup
Subject: six attendants in related practical garments: indigo work coats, translucent rain capes, black mourning ribbons, brass valve badges, weather gloves, notebook satchels, tether belts, one with a rolled patch kit, one with a signal-orange trim reserved for release crew
Style/medium: realistic costume lineup photography
Composition/framing: full-body straight-on lineup with even spacing, neutral poses, each figure clearly visible, restrained prop use
Lighting/mood: soft overcast daylight, quiet institutional dignity
Color palette: slate blue, soot black, ivory, tarnished brass, one faded orange accent
Materials/textures: waxed cotton, translucent plastic rain capes, wool scarves, leather gloves, brass clips, canvas satchels
Constraints: practical service uniforms; no runway glamor; no crowns; no military medals; no logos; no watermark; figures should read as workers and caretakers
Avoid: luxury fashion campaign, cosplay, superhero silhouettes, theatrical makeup, beauty-ad retouching
