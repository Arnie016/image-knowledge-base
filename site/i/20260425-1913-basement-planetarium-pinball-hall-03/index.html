Prompt Extract

Use case: ui-mockup
Asset type: game HUD reference frame
Primary request: a realistic pinball playfield interface view for a fictional machine called Orbit House, showing the active table with chrome rails, orbit lanes, moon bumpers, comet targets, a compact score cluster, ball count, bonus ladder, and a small constellation mission strip
Scene/background: the camera looks slightly down onto the glowing playfield surface inside a real mechanical pinball cabinet, with the backglass visible at the top edge and physical table art dominating the frame
Subject: a themed browser-game-like HUD translated into a believable pinball score surface that protects the playfield and keeps the center readable
Style/medium: realistic game UI mockup blended with photographed pinball hardware
Composition/framing: portrait-oriented full table view, one compact status cluster, one narrow secondary strip, no large overlay panels
Lighting/mood: incandescent arcade glow with cool star-map highlights, focused and playable
Color palette: navy, cream, chrome, cherry red, amber, pale cyan
Materials/textures: glass reflections, painted wood, chrome rails, printed score inserts, Bakelite buttons, translucent plastics
Constraints: no logos, no watermark, minimal short labels only, no generic dashboard layout, no center-obscuring overlay, must feel mechanical and playable
Avoid: SaaS admin panel styling, espor
